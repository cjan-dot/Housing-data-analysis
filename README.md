# Housing-data-analysis

## Overview
This project analyses an Ames Housing dataset using Python to identify factors that influence residential property prices.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Tasks
- Data cleaning and preprocessing
- Missing value handling
- Feature engineering
- Logarithmic transformation
- Correlation analysis
- Data visualisation

## Dataset
The dataset contains 2,930 residential property observations and 82 housing attributes, including property characteristics, location information, and sale prices.

## Key Findings
- Overall house quality demonstrated one of the strongest relationships with sale price.
- Larger living areas were generally associated with higher property values.
- Newer properties tended to achieve higher sale prices than older properties.
- Housing observations were unevenly distributed across neighbourhoods.
- One- and two-storey houses represented the majority of properties within the dataset.
